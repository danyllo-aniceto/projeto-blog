O que é um banco de dados

Um banco de dados é uma coleção organizada de informações ou dados estruturados,normalmente armazenados eletronicamente em um sistema de um desktop .O banco de dados é geralmente usado por um sistema de gerenciamento de banco de dados(DBMS).

Os dados são modelados em linhas e colunas em uma série de tabelas para tornar o processamento e a consulta de dados eficientes.Os dados podem ser facilmente acessados e gerenciados.A maioria dos bancos de dados usa a linguagem de consulta estruturada (SQL) para escrever e executar dados.
